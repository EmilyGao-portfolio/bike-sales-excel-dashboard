# Bike Sales Customer Analysis
## Project Overview
This Excel project analyzes customer characteristics associated with bike purchases.
## Business Questions
1. Which customer demographic factors are associated with bike purchases?
2. How do income, age, commute distance, and region influence purchasing behavior?
## Tools
- Microsoft Excel
- PivotTables
- PivotCharts
- Slicers
- Data cleaning

## Key Insights
- Bike purchasing patterns varied by income and customer age group.
- Shorter commute distances showed different purchasing patterns from longer commutes.
- Dashboard slicers allow users to compare results by marital status, region, and education.
